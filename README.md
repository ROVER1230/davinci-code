## 🎯 游戏简介

达芬奇密码是一款经典的推理猜牌桌游。你和对手各自拥有几张按**从小到大顺序排列**的隐藏牌，通过**轮流猜测对手的牌**来逐步揭开对方的秘密。

**你的目标**：在自己所有牌被猜出之前，猜出对手所有的牌！

### 核心规则

| 规则 | 说明 |
|------|------|
| 🃏 **牌组** | 26张牌：⚫黑色 0-11（12张）+ ⚪白色 0-11（12张）+ 黑白鬼牌各1张（2张） |
| 🤚 **起始手牌** | 2-3人各4张，4人各3张 |
| 📐 **排列规则** | 从左到右数字由小到大，同数字黑色 < 白色，鬼牌可放任意位置 |
| 🔄 **回合流程** | 摸牌 → 猜测 → 猜对可继续/猜错回合结束 |
| 🏆 **胜利条件** | 成为唯一还有未公开牌的玩家 |

### 详细流程

1. **摸牌**：从牌堆摸1张牌（仅自己可见）
2. **猜测**：猜任意对手任意一张牌的颜色+数字（如"⚫黑5"）
3. **猜对** → 对手摊开该牌，你可选择继续猜或停止（将摸到的牌秘密插入手牌）
4. **猜错** → 你摸到的牌公开并按顺序插入你的手牌，回合结束
5. **鬼牌**：摸到鬼牌需选择放置位置，猜测时说"黑色鬼牌"或"白色鬼牌"
